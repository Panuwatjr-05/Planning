# Product

## Register

product

## Users

นักศึกษาและคนทำงาน ที่มีหลาย project, goal, และไอเดียในหัว แต่มักลืมหรือไม่ได้ลงมือทำ ใช้แอปนี้ระหว่างวัน บนโต๊ะเรียนหรือโต๊ะทำงาน เพื่อ capture สิ่งที่ต้องทำและติดตาม progress ของชีวิต

## Product Purpose

ระบบจัดการแผนและเป้าหมายส่วนตัว แก้ปัญหา "มีแผนในหัวแต่ลืมหรือไม่ได้ทำ" ด้วยการให้ capture ได้เร็ว ไม่มี friction และทบทวนได้ง่าย ความสำเร็จคือผู้ใช้รู้สึกว่าชีวิตมีทิศทาง และสิ่งที่คิดไว้ไม่หายไป

## Brand Personality

สงบ เป็นมิตร สะอาด — เหมือนสมุดจดที่ดีที่สุดที่เคยมี ไม่กดดัน ไม่วุ่นวาย แต่มีเอกลักษณ์พอที่จะจำได้

Reference: Notion — calm, minimal, feels like a tool that trusts its user

## Anti-references

- **Jira / Asana** — cluttered, enterprise-heavy, ไม่ใช่แอปส่วนตัว
- **Dark / moody UI** — heavy dark mode ที่รู้สึกหนัก ไม่ใช่ feeling ที่ต้องการ
- **Over-gamified productivity apps** — badges, streaks, points ที่มากเกินไปจนดูเหมือน game ไม่ใช่ tool

## Design Principles

1. **Capture first, organize later** — ทุก flow ต้องบันทึกได้ใน 1 วินาที ไม่มีขั้นตอนบังคับก่อน save
2. **Calm confidence** — UI ไม่ส่งเสียง ไม่กดดัน แต่ให้ความรู้สึกว่าทุกอย่างอยู่ในมือ
3. **Earn every pixel** — ไม่มี element ที่ไม่มีความหมาย ทุก decoration ต้องช่วย comprehension หรือ delight
4. **Personal, not corporate** — tone เป็นกันเอง ภาษาไทยเป็นธรรมชาติ ไม่ใช่ซอฟต์แวร์องค์กร
5. **Reward completion** — celebration เมื่อ goal สำเร็จต้องรู้สึก rewarding จริงๆ ไม่ใช่แค่ checkbox

## Accessibility & Inclusion

- ภาษาไทยเป็น primary language — font ต้องรองรับ Thai script ครบถ้วน
- WCAG AA minimum — contrast ratio ผ่านมาตรฐาน
- Light mode only (ตามที่ผู้ใช้เลือก) — ไม่มี dark mode heavy
